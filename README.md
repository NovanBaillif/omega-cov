# omega-cov

A token-level metric for detecting confabulation in LLM outputs via covariance
of cosine displacement and Shannon surprise.

## What it does

`omega-cov` computes three quantities on a sequence of tokens generated by an
open-weight LLM:

- **A_marginal** = mean(δ) × mean(σ) — the product of mean cosine displacement
  between consecutive hidden states and mean Shannon entropy per token.
- **A_joint** = mean(δ · σ) — the mean of the token-by-token product.
- **A_cov** = A_joint − A_marginal — the covariance between displacement and
  surprise, which discriminates regimes that A_marginal cannot.

The signature classification (`ALIGNED`, `MIXED`, `ANTI`) flags regions where
surprise and displacement covary (signal) versus regions where surprise occurs
without semantic movement (candidate confabulation).

## Why

Standard metrics measure surprise (perplexity) and displacement (cosine drift)
separately, or product their averages. Two sequences with identical mean
surprise and mean displacement can have opposite token-by-token dynamics — one
where each surprising token moves meaning, one where surprise is stylistic and
no real movement occurs. `A_cov` separates these cases in a single forward
pass, without sampling multiple generations or training a probe.

## Quick start

```python
from omega_cov import measure

result = measure(
    "The text to analyze.",
    model="mistralai/Mistral-7B-v0.1",
)

print(result.A_marginal)   # 2.0345
print(result.A_joint)      # 2.1102
print(result.A_cov)        # 0.0757
print(result.signature)    # "ALIGNED"
```

For batch processing, load the model once:

```python
from omega_cov import OmegaCov

oc = OmegaCov(model="mistralai/Mistral-7B-v0.1")
for text in corpus:
    result = oc.measure(text)
    print(result.A_cov, result.signature)
```

## Requirements

- Python ≥ 3.9
- An open-weight LLM that exposes hidden states and logits (Mistral, Llama,
  Qwen, Gemma, etc.). API-only models like GPT-4 or Claude are not supported
  because they do not expose these.
- GPU recommended for models > 1B parameters.

## Status

Alpha. Default thresholds for `ALIGNED` / `MIXED` / `ANTI` (currently ±0.1)
will be calibrated against TriviaQA and WikiBio corpora before v1.0.

## Limitations

- Does not detect *consistent hallucinations* (the model always says X and X
  is wrong). `A_cov` measures dynamic coherence, not factual truth.
- Requires access to logits and hidden states, ruling out closed-API models.
- Cross-substrate stability tested on Mistral-7B and TinyLlama-1.1B
  (Pearson r ≈ 0.95 on 10 segments). Broader evaluation pending.
